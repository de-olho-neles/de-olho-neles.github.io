De Olho Neles

Essa é uma aplicação web com o objetivo de facilitar o acesso do grande público às informações de gastos da API da camara dos deputados, assim como gerar um resumo a cada 1 ou 2 semanas de todos os gastos feitos pelos deputados que o usuário decidir monitorar.

Caso você queria contribuir com o projeto, temos um repositório público no GitHub.
